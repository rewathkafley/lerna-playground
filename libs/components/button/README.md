# Button Component

## Usage


### Importing
e.g. in `app.module.ts`
```
 import SfButtonModule from '@safire/components/button';
 
 
 @NgModule({
 imports: [SfButtonModule]
 ...
 })
 export class AppModule {
 
 }
```

### Adding to view

e.g. in `app.component.html`

```html
 <div class="container">
 
<button type="button" sf-button primary>Click me</button>
 
 </div>
```
